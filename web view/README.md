# Saily Web View

This is a standalone web preview inspired by Instagram Web, styled with your Saily theme.

## Quick run

- Open `index.html` directly in a browser
- Or run a local static server from this folder

Example (PowerShell):

```powershell
cd "S:/new_app/web view"
python -m http.server 5500
```

Then open: [http://localhost:5500](http://localhost:5500)
